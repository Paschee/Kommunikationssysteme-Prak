Package tests.test2

Grundlage für Versuchsserie 2
Der Client sendet einen HTTP GET-Request an einen HTTP-Dienst (Web-Server).
Der Server ist ohne Funktion, soll zum HTTP-Dienst ergänzt werden, der ausschließlich das HTTP-KOmmando "GET" bearbeiten kann.

Beide "Geräte" müssen mit der MAC- und IP-Adresse des Netzwerkinterfaces Ihres Rechners konfiguriert werden, das für die Versuche verwendet wird (z.B. WLAN oder Kabel).
Das Protokoll ARP ist implentiert, so das die MAC-Adressen der anderen Geräte nicht manuell bestimmt werden müssen.
Eigene Adressinformationen werden vom Kommando "ifconfig -a" geliefert.
Die IP-Adresse der Webservers wird durch die Kommandos "host hostname" oder "ping hostname" geliefert.
